# Clipzioo — YouTube Downloader & Video Trimmer

A premium, fast, and secure YouTube downloader and clipper. Extract high-quality MP4 video clips or MP3 audio from any YouTube video instantly.

### 🌐 Live Production URL
**[https://clipzioo.com](https://clipzioo.com)**

---

## Features
- **Video Downloads**: High-quality MP4 downloads up to 1080p.
- **Audio Extraction**: Premium MP3 conversion with metadata and thumbnails.
- **Video Trimming**: Select custom start and end times to download specific clips.
- **Playlist Support**: Detection and fetching of YouTube playlists.
- **History**: Track your recent downloads locally.

## Tech Stack
- **Frontend**: HTML5, Vanilla JS, Inter Typography.
- **Backend**: Node.js (Express), yt-dlp, FFmpeg.
- **Desktop**: Electron integration.

## Setup
1. Clone the repository.
2. Install dependencies: `npm install`
3. Create a `.env` file with your `API_URL`.
4. Run the app: `npm start` (Backend) or `npm run app` (Electron).
